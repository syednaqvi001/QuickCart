🚀 HOW TO RUN
================================================================================

QUICK START (5 MINUTES)
1. cd QuickCart
2. cp .env.example .env
3. docker-compose up --build
4. Wait 2-3 minutes for services to be healthy
5. Test: curl http://localhost/api/auth/register -d {...}

VIEW LOGS
- All services: docker-compose logs -f
- Specific service: docker-compose logs -f auth-service

STOP SERVICES
- Stop: docker-compose down
- Remove data: docker-compose down -v

ACCESS SERVICES
- Frontend: http://localhost:3000
- API Gateway: http://localhost
- Auth Service: http://localhost:3001
- RabbitMQ: http://localhost:15672 (guest/guest)
- MinIO: http://localhost:9001 (minioadmin/minioadmin)
- PostgreSQL: localhost:5432 (quickcart/quickcart123)
- Redis: localhost:6379
